## 安装问题

### Q: 插件无法安装，提示格式不正确

**可能原因**：
- 打包文件损坏
- 文件格式不对

**解决方法**：
1. 确保使用 `python setup.py sdist --formats=zip` 打包
2. 不要解压再重新压缩，直接使用生成的文件
3. 确认 Python 版本在 3.6 以上

### Q: Video Station 无法识别插件

**可能原因**：
- INFO 文件有问题
- 插件 ID 不正确

**解决方法**：
1. 检查 `INFO` 文件是否正确生成
2. 运行 `make install-test` 测试
3. 确认插件名称（目录名）正确

## 配置问题

### Q: 配置页面无法访问

**可能原因**：
- 配置服务未启动
- 防火墙阻止连接
- 端口被占用

**解决方法**：
1. 在 Video Station 中点击"测试连接"重启服务
2. 确认端口 5125 未被占用
3. 检查 NAS 防火墙设置
4. 尝试 `http://[NAS_IP]:5125`（不要加 https）

### Q: 配置保存后不生效

**可能原因**：
- 保存失败
- 缓存问题

**解决方法**：
1. 刷新配置页面
2. 检查配置文件权限
3. 在 Video Station 中重新测试连接

### Q: Cookie 配置无效

**可能原因**：
- Cookie 过期
- Cookie 格式不对

**解决方法**：
1. 重新从浏览器获取最新的 Cookie
2. 确保没有多余的空格或换行
3. 检查是否需要其他认证方式

## 刮削问题

### Q: 找不到任何元数据

**可能原因**：
- 刮削源配置问题
- 网络问题
- 网站不可访问

**解决方法**：
1. 确认网络连接正常
2. 在配置页面检查启用的刮削源
3. 尝试使用其他刮削源
4. 用 `--loglevel debug` 查看详细日志

### Q: 元数据错误或不完整

**可能原因**：
- 网站结构变化
- 刮削源需要更新

**解决方法**：
1. 更新到最新版本
2. 检查其他刮削源是否能正常工作
3. 手动修正元数据

### Q: 某特定网站返回 403 错误

**可能原因**：
- 反爬机制
- IP 被封
- 需要 Cookie

**解决方法**：
1. 配置该网站的 Cookie
2. 启用 DNS over HTTPS（如果有配置）
3. 检查该网站在浏览器是否能正常访问
4. 尝试其他刮削源

### Q: 刮削速度很慢

**可能原因**：
- 网络问题
- 启用太多刮削源

**解决方法**：
1. 减少同时启用的刮削源
2. 调整优先级，把常用的设为高优先级
3. 检查网络连接
4. 考虑使用 API 类型的刮削源

## 性能问题

### Q: 刮削需要很长时间

**解决方法**：
- 减少 `--limit` 参数
- 只启用必要的刮削源
- 设置更高优先级的刮削源先执行

### Q: 配置服务器响应慢

**解决方法**：
- 重启配置服务
- 检查 NAS 资源使用情况

## 开发问题

### Q: 如何调试刮削源

**解决方法**：
```bash
python main.py --type movie --input "{\"title\":\"测试\"}" --limit 1 --loglevel debug
```

### Q: JSON 验证失败

**解决方法**：
```bash
# 使用验证脚本
python scripts/validate_flows.py

# 或者使用在线工具
# https://jsonlint.com/
```

### Q: 如何测试新创建的刮削源

**解决方法**：
1. 先用 `make validate` 验证
2. 用 `--loglevel debug` 测试
3. 检查输出是否符合预期

### Q: 找不到 `[plugin_id]` 替换后的结果

**解决方法**：
`[plugin_id]` 会被替换为实际的插件 ID（目录名）。如果使用手动测试，需要确认目录名是什么。

## 其他问题

### Q: 如何请求添加新的刮削源

**解决方法**：
1. 在 GitHub 上提交 Issue
2. 描述你想要添加的网站
3. 如果可能，提供示例 URL 和格式

### Q: 插件安全吗

**回答**：
- 所有代码开源，可以检查
- 只做读取操作，不会修改数据
- 配置信息存在本地
- 只会连接你启用的网站

### Q: 支持哪些语言

**回答**：
- 主要支持中文（简体/繁体）
- 部分刮削源支持英文、日文
- 查看具体刮削源的 `lang` 字段

### Q: 如何备份配置

**解决方法**：
- 备份 `scrapeflows.conf` 文件
- 备份 `resolvers.conf` 文件（如果有修改）
